# Logs Observability

Logs provide the unstructured signal required for correlation and diagnostics. 

## Central Log Store 
Elasticsearch or OpenSearch is used ad the central logging backend. 

## Logged Features
- Error density and error rate curves
- Burst detection 
- Rare pattern discovery 
- Log template clustering 
- Service-to-Service causality signals 

These features will feed into the AI training pipeline.
