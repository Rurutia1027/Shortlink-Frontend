# SQL Observability 
SQL performance is a major contributor to microservice latency.

## Collected Metrics 
- Query latency histograms (p50/p90/p99)
- Slow query logs 
- Lock wait times 
- Transaction retries 
- Error classification 

## Dashboards 
- Slow Query Top-N
- Query latency distribution 
- Lock contention heatmap 
- Transaction throughput and error rate

## Future Extension 
Raw SQL logs will be fed into the AI pipeline to train: 
- Slow-query anomaly models 
- Query-template clustering
- Execution-pattern anomaly models
