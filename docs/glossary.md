# Glosaary 

A reference of terms and used across metrics, logs, tracing, JVM, SQL, and AI observability systems. 

Includes: 
- SLI, SLO, SLA
- TSDB, retention, downsampling 
- Heap, GC, thread pools, safepoints
- SQL latency histogram 
- ElasticSearch index 
- Feature store 
- Anomaly, residual, correlation window
